MatchBox thinking inside the box

Help

MatchBox helps you find a box.

Searching

Use the search boxes to enter numerical values for height, width or depth. The application will return all boxes larger than the dimension entered. All dimensions are measured in inches.

If you see a check mark next to box name, that indicates a common box size that we likely have in stock.

Sorting

You can sort multiple columns by holding down the shift key.

This is a demonstration version of Matchbox which pulls from a static data file. For instructions on how to set up the application to pull data dynamically from ArchivesSpace, please visit the GitHub repository.
